# Readme with Privacy Policy and Terms of Service for OURA Developer Application

## Overview
This repository is for a personal project that connects to the Oura Ring API to access and analyze health data such as sleep, activity, and readiness. The project is developed and maintained solely for personal, non-commercial use.

---

## Privacy Policy

**Last updated:** November 3, 2025

This application connects to the Oura API to retrieve personal data from the user’s Oura Ring account. The App is developed and maintained by an individual developer for private, non-commercial purposes.

### Data Collection and Usage
- The Application accesses only the data explicitly authorized by the user through Oura’s OAuth 2.0 flow.
- Data is used exclusively for personal analytics and visualization.
- No data is shared, sold, or transmitted to third parties.

### Data Storage
- Data may be stored locally (e.g., on a Raspberry Pi, local computer, or private database) for analysis.
- No data is hosted on public servers or cloud services unless explicitly configured by the developer.

### Data Deletion
- Since the App is for personal use, the developer can delete all locally stored data at any time.
